# TASK
In general, we want to get into the space of .io games.

The task in here is to build a simple multiplayer game. 

My preferences:

- Game Engine: phaser
- Typescript
- For the backend, either fastify or Nest.js
- If we need a monorepo, use nx (or choose somethign different if you think it's overkill)
- If we need a db, postgres or sqlite
- We want a game that does not take forever to create, is multiplayer, and is actually fun. 
- I guess websockets as a communication protocol is the way to go, if you think otherwise, please interfere
- Create!
